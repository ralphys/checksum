
# Checksum
## License ![License](https://img.shields.io/badge/license-GPLv2-green.svg)

A GUI tool to easily calculate MD5, SHA-1, SHA-224, SHA-256, SHA-348, SHA-512 files hashes in Linux Lite / Ubuntu based systems.
